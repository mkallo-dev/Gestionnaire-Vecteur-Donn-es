# Array Analyst C++

## 📌 Description
Ce projet est un exercice d'algorithmique en C++ conçu pour démontrer une gestion rigoureuse des structures de données statiques et des flux d'entrée/sortie. Le programme permet de saisir un tableau d'entiers avec validation de données, d'afficher son contenu et de calculer des statistiques clés (somme et produit).

## 🚀 Fonctionnalités
- **Saisie Sécurisée** : Validation des entrées utilisateur pour éviter les erreurs de type.
- **Gestion des Limites** : Contrôle du débordement de mémoire (Buffer Overflow).
- **Analyse de Données** : Algorithmes de calcul de somme et de produit.
- **Code Propre** : Utilisation de constantes et passage de paramètres par référence pour l'optimisation.

## 🛠️ Concepts Techniques Appliqués
- Séparation des préoccupations (Modularité via fonctions).
- Utilisation de `const` pour la sécurité des données.
- Gestion du flux `std::cin` (vidage du tampon en cas d'erreur).
- Programmation structurée conforme aux standards C++ modernes.
